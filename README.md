# Harsh Saw - Software Engineer Portfolio

Welcome to my personal portfolio! This repository houses the source code for my portfolio website, visualizing my experience, projects, and technical skills using modern web technologies.

**Live Site:** [harshsaw.me](https://harshsaw.me)

## 🚀 Tech Stack

- **Framework:** [Next.js 15](https://nextjs.org/) (React)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** GSAP, Framer Motion
- **3D Graphics:** OGL
- **Smooth Scrolling:** Lenis
- **Deployment:** Vercel

## 📂 Project Structure

```
src/
├── app/                  # Next.js App Router pages
├── assets/               # Static assets (images, icons)
├── components/           # Reusable React components
│   ├── sections/         # Page sections (Hero, Projects, Experience, etc.)
│   ├── shared/           # Shared UI components (Buttons, Cards, etc.)
│   └── ui/               # Base UI elements
├── data/                 # Data files (projects, experiences, skills, profile)
└── hooks/                # Custom React hooks
```

## 🛠️ Features

- **Performance First:** Optimized for speed and SEO with Server Components and efficient rendering.
- **Interactive UI:** Smooth scrolling, 3D elements, and engaging animations.
- **Dynamic Content:** Data-driven sections for easy updates to projects and experience.
- **Responsive Design:** Fully responsive layout for all devices.
- **Modern Design:** Clean, minimalist aesthetic with a focus on typography and visual hierarchy.

## 🏃‍♂️ Running Locally

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Harshksaw/Portfolio.git
    cd Portfolio
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  Open [http://localhost:3000](http://localhost:3000) with your browser.
