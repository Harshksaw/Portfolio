# Portfolio Repository Context

## Summary
This repository is the source code for the portfolio website itself. It is separate from the portfolio owner's personal skill set and project history, but it provides useful technical context about how the portfolio is built and maintained.

## Live Site
- https://harshsaw.me

## Portfolio Site Stack
- Next.js 15
- React
- TypeScript
- Tailwind CSS
- GSAP
- Framer Motion
- OGL
- Lenis
- Vercel

## Repository Structure Summary
- `src/app`: Next.js App Router pages
- `src/assets`: Static assets
- `src/components`: Reusable React components
- `src/components/sections`: Main portfolio sections
- `src/components/shared`: Shared UI components
- `src/data`: Source data for profile, projects, experience, and skills

## Portfolio Site Features
- Performance-focused rendering and SEO setup
- Interactive UI with animations and 3D elements
- Data-driven sections for easy updates
- Responsive design across devices
- Visual emphasis on typography and hierarchy
